HTML Tags

Notes:

Highlighting Text
<mark></mark> will highlight important or referenced text.
Superscripts
<sup></sup> will turn regular text into superscripts.
Abbreviations
<abbr title=""></abbr> will abbreviate text that is hovered over. Typically used for acronyms, for example, "HTML and HTTP."
Links
<a href=""></a> will add links to text. By adding target="_blank" into the tag, it will cause the links to open in a new tab.
Adding Lists
There are a variety of different list tags. There is the <ul></ul> tag for unordered lists, <ol></ol> for ordered lists, and <dl></dl> for description lists. Unordered lists can be used for text that is not meant to appear in any specific order (for example; ingredients list, christmas shopping list, list of different phone brands.) Ordered lists can be used for text that is required to be in a specific order (example; top musicians of 2022, most expensive cars, most viewed youtube videos.) Description lists have a <dt></dt> (description term) and <dd></dd> (description description) tag inside. This can be useful when listing information about items, values, or data.
